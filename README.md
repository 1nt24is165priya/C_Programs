##Topics Coverd
- Sorting algorithms
  
##Programs:
### 1.Selection Sort
- Code and Time Complexity calculation
- Time Complexity of selection sort O(n^2)
- Space Complexity of selection sort O(1)
  
### 2.Insertion Sort
- Code and Time Complexity calculation
- TC(Best case) = O(n) , When Only 1 comparisions per element,no shifting needed.
- TC(worst and average case) = O(n^2)
- Space Complexity = O(1)

### 3.Merge Sort
-Code, Time and Space complexity calculation
-TC(Best,Avg,Worst) = O(nlogn)
-Space Complexity = O(n) [for tempo array(used in merging)]

### 4.Quick Sort
-Code ,Time and space complexity calculation
-TC
- Best Case:
Pivot divides array evenly → O(nlog n)
- Average Case:
Random pivot distribution → O(nlog n)
- Worst Case:
Pivot always smallest/largest (already sorted array) → O(n^2)

-Space Complexity
- Space Complexity: O(log n) due to recursion stack (in‑place sorting, no extra arrays needed).
